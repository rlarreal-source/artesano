# 🌿 Artesano — Sistema de Gestión de Ventas

App web para gestión de ventas de Artesano. Funciona en iPhone, Android y computador.

## Cómo usar

Visita la URL de GitHub Pages de este repositorio.

## Para agregar a iPhone como app

1. Abre la URL en Safari
2. Toca el botón compartir (□↑)
3. Selecciona "Agregar a pantalla de inicio"
4. Toca "Agregar"

La app aparecerá en tu pantalla de inicio como cualquier otra app.
